# My Portfolio

This is a small personal portfolio website containing `index.html`, `style.css`, and `script.js`.

How to preview

- Open `index.html` in your browser.
- Or serve the folder with Python (runs a simple HTTP server):

```powershell
cd "c:\Users\Lenovo\OneDrive\Desktop\portfolio"
python -m http.server 8000
# then open http://localhost:8000
```
